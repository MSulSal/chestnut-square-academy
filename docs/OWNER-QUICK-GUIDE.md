# Owner Quick Guide (Non-Technical)

This guide is for daily website use after handoff.

## Most Common Tasks
1. Edit page text:
- Go to `Pages`
- Hover a page and click `Edit with Elementor`
- Update text or images
- Click `Update`
2. Change phone/email/hours everywhere:
- Go to `Settings > CSA Business Profile`
- Update fields once
- Save changes
3. Check tour requests:
- Go to `Tour Requests`
- Open newest entries
4. Verify launch readiness:
- Go to `Tools > CSA Launch Kit`
- Confirm blocking items are `0`
5. Follow owner data map:
- Go to `Tools > CSA Launch Kit > Owner Data Entry Map (With Examples)`
- Fill fields from `Settings > CSA Business Profile` and `Settings > CSA Tour Form`

## Before Any Public Update
1. Make sure new information is confirmed (no placeholders).
2. Save a backup (UpdraftPlus).
3. Preview page on mobile and desktop.
4. Confirm contact details are still correct.

## Staging vs Production
- Staging mode (`Noindex`) is for private work.
- Production indexing is for public launch.
- Switch mode in `Tools > CSA Launch Kit`.

## If Something Looks Wrong
1. Check `Tools > CSA Launch Kit` for blockers.
2. Confirm required plugins are active.
3. Restore from latest backup if needed.
4. Contact your technical support person.

## Keep These Updated
- Phone number
- Public email
- Business hours
- Address/map pin
- Program availability details
- FAQ answers

## Monthly Routine (15 Minutes)
1. Submit a test tour form and confirm email receipt.
2. Verify Google Business Profile points to the correct website.
3. Confirm no expired or incorrect announcements.
4. Download a fresh preflight report for records.
